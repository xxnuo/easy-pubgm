# Easy PUBG Mobile

基于 [AutoJS6](https://github.com/SuperMonster003/AutoJs6) 的`和平精英`全自动掉分`脚本`。

无需 Root，无需适配机型。

## V2 自动跳伞往圈外跑版本

### 原理

- 跳伞后回头滑动，让视角移动到飞机航线相反方向
- 循环推动左摇杆前进
- 等待死亡，死亡后自动退出游戏
- 开始游戏，继续循环

### 使用前设置

安装 [AutoJS6](https://github.com/SuperMonster003/AutoJs6/releases) 并授予相关权限。

- 必须开启无障碍服务按按钮
- 屏幕录制权限用于 OCR 识别各种按钮
- 其他按需设置，比如悬浮窗方便启动和停止脚本

### 游戏设置

- **地图设置**：排位模式，关闭匹配队友！地图选择四周是海域的：
  - 主题海岛（航线没有纯海岛随机，比如创乐园起始点一般是海）
  - 度假岛
  - 雨林
  - 雪地
  - 黄金岛

  这五张地图匹配，选的多匹配的快，其他的比如`纯海岛`、`荣都`、`沙漠`都有可能被障碍物挡住，需要等待毒死，不推荐
- **画面设置**：流畅画质，低帧数（降低发热并提升 OCR 识别速度）
- **操作设置**：操作方式是左侧摇杆，右侧视角，左侧摇杆和冲刺拉到最底部最好，方便长时间冲刺
- **跳伞设置**：将离开按钮透明度设置为位于屏幕左下部分且完全不透明（便于 OCR 识别自动跳伞）
- **其他设置**：关闭任何邀请，防止被人邀请影响识别

### 使用方法

- 下载 [easypubgmv2.js](easypubgmv2.js) 脚本
- 打开游戏，进入特训岛
- 在 AutoJS6 的悬浮窗中选择三个点的菜单，打开指针位置显示功能
- 然后在特训岛测出一些脚本内需要的坐标，然后修改脚本内这几行

```javascript

// 跳伞后回头滑动的起始坐标和结束坐标，可在特训岛打开指针位置然后将视角从南移到北测试得到，需要耐心多次调试得到合适的数值
const TURN_BACK_START_POINT = [2760, 835];
const TURN_BACK_END_POINT = [2305, 842];

// 跳伞后回头完往天上看的滑动的起始坐标和结束坐标，让跳伞飞的更远
const LOOP_UP_START_POINT = [2543, 1209];
const LOOP_UP_END_POINT = [2567, 531];

// 将左摇杆从起始点移动到冲刺点，让角色冲刺飞行
const SPRINT_START_POINT = [703, 1417];
const SPRINT_END_POINT = [703, 30];

```

- 调整完成后，在 AutoJS6 的悬浮窗中选择三个点的菜单，关闭指针位置显示功能
- 退出特训岛，在大厅开启你编辑好的脚本，然后开始挂机！

Enjoy! 享受全自动掉分吧!

## V1 跳伞后大退版本

> 废弃版本，游戏更新后被制裁了，提示“您因多次在竞赛中逃跑，被临时禁止参与经典模式匹配”会被禁赛三十分钟

目前逻辑较为简单：跳伞后大退。不能保证稳定把把掉大分但能简单快速掉分。

### 原理

- 自动打开和平精英
- 自动关闭公告/活动/重连弹窗
- 自动关闭匹配队友选项并开始游戏
- 等待跳伞后自动关闭游戏
- 循环执行以上流程

### 使用前设置

安装 [AutoJS6](https://github.com/SuperMonster003/AutoJs6/releases) 并授予相关权限。

- 必须开启无障碍服务按按钮
- 屏幕录制权限用于 OCR 识别各种按钮
- Shizuku 权限用于杀掉游戏进程
- 其他按需设置

在游戏内进行以下设置：

- **画面设置**：流畅画质，低帧数（降低发热并提升 OCR 识别速度）
- **跳伞设置**：将离开按钮透明度设置为完全不透明（便于 OCR 识别自动跳伞）
- **其他设置**：海岛模式掉分较多一点，容都更快但是掉分较少

### 使用方法

- 下载 [easypubgmv1.js](easypubgmv1.js) 脚本
- 在 AutoJS6 中打开脚本运行即可
