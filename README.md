## V1 跳伞后大退版本

> 废弃版本，游戏更新后被制裁了，提示“您因多次在竞赛中逃跑，被临时禁止参与经典模式匹配”会被禁赛三十分钟

基于 [AutoJS6](https://github.com/SuperMonster003/AutoJs6) 的和平精英全自动掉分脚本。
目前逻辑较为简单：跳伞后大退。不能保证稳定把把掉大分但能简单快速掉分。

### 功能

- 自动打开和平精英
- 自动关闭公告/活动/重连弹窗
- 自动关闭匹配队友选项并开始游戏
- 等待跳伞后自动关闭游戏
- 循环执行以上流程

### 使用前设置

安装 [AutoJS6](https://github.com/SuperMonster003/AutoJs6) 并授予相关权限。

- 屏幕录制权限用于 OCR 识别各种按钮
- Shizuku 权限用于杀掉游戏进程
- 其他按需设置

在游戏内进行以下设置：

- **画面设置**：流畅画质，低帧数（降低发热并提升 OCR 识别速度）
- **跳伞设置**：将离开按钮透明度设置为完全不透明（便于 OCR 识别自动跳伞）
- **其他设置**：海岛模式掉分较多一点，容都更快但是掉分较少

### 使用方法

- 下载 `easypubgmv1.js` 脚本
- 在 AutoJS6 中打开脚本运行即可
